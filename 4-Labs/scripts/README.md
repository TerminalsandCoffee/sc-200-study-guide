# Automation Scripts

PowerShell, Azure CLI, and other automation helpers for Microsoft security operations.

## Structure

- `powershell/` - PowerShell scripts for Sentinel, Defender, and Azure
- `azure-cli/` - Azure CLI scripts and commands
- `python/` - Python scripts for API interactions
- `utilities/` - Helper scripts and one-liners

## Script Categories

### Sentinel Management
- Bulk rule deployment
- Data connector configuration
- Workbook management
- Query testing and validation

### Defender Operations
- Defender for Cloud configuration
- M365 Defender API interactions
- Defender for Endpoint management
- Threat intelligence management

### Data Operations
- Log ingestion testing
- Data export and backup
- Query performance optimization
- Data source validation

### Automation
- Scheduled tasks
- CI/CD for detection rules
- Bulk operations
- Configuration management

## Best Practices

- Include error handling
- Use managed identities where possible
- Document parameters and usage
- Add logging and verbose output
- Test in non-production first
