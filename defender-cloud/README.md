# Microsoft Defender for Cloud

Microsoft Defender for Cloud security posture management, recommendations, and threat protection.

## Structure

- `security-posture/` - Cloud Security Posture Management (CSPM)
- `workload-protection/` - Cloud Workload Protection Platform (CWPP)
- `recommendations/` - Security recommendations and remediation
- `alerts/` - Security alerts and threat detection
- `secure-score/` - Secure Score tracking and improvement

## Key Topics

### Security Posture
- Cloud Security Posture Management (CSPM)
- Security recommendations
- Compliance assessments
- Secure Score
- Regulatory compliance (NIST, PCI-DSS, etc.)

### Workload Protection
- Servers (Defender for Servers)
- Containers (Defender for Containers)
- Storage (Defender for Storage)
- SQL (Defender for SQL)
- App Service (Defender for App Service)
- Key Vault (Defender for Key Vault)
- Resource Manager (Defender for Resource Manager)
- DNS (Defender for DNS)

### Threat Protection
- Security alerts
- Just-in-Time (JIT) VM access
- Adaptive application controls
- File integrity monitoring
- Network security recommendations

## Exam Focus

- Understanding CSPM vs CWPP
- Security recommendations and remediation
