# Jupyter Notebooks

Advanced analysis notebooks for threat hunting, investigation, and data science in Microsoft Sentinel.

## Structure

- `hunting/` - Threat hunting notebooks
- `investigation/` - Incident investigation notebooks
- `ml-analysis/` - Machine learning and anomaly detection
- `data-analysis/` - Data exploration and analysis

## Use Cases

- Complex multi-step investigations
- Machine learning model development
- Advanced data correlation
- Custom visualizations
- Statistical analysis
- Behavioral profiling

## Notebook Format

Each notebook should include:
- Objective and context
- Data source information
- Step-by-step analysis
- Visualizations
- Conclusions and next steps

## Requirements

- Azure ML workspace or local Jupyter environment
- Sentinel API access
- Required Python libraries (pandas, matplotlib, etc.)
- Authentication configured
