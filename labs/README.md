# Hands-On Labs

Practical, hands-on scenarios for Microsoft Sentinel, Defender for Cloud, and Microsoft 365 Defender.

## Structure

- `sentinel-labs/` - Sentinel configuration, data connectors, and scenarios
- `defender-cloud-labs/` - Defender for Cloud security posture exercises
- `m365-defender-labs/` - Email, Identity, and Cloud Apps scenarios
- `defender-endpoint-labs/` - EDR and endpoint protection labs
- `incident-response-labs/` - End-to-end incident response scenarios
- `automation-labs/` - Logic Apps and playbook development

## Lab Format

Each lab should include:
- **Objective** - What you'll learn
- **Prerequisites** - Required setup and permissions
- **Scenario** - The attack or security event
- **Steps** - Detailed walkthrough
- **Expected Results** - What you should see
- **Key Takeaways** - Important concepts learned

## Lab Ideas

- Configure data connectors (Syslog, CEF, Windows Security Events)
- Create custom analytics rules
- Build incident response playbooks
- Set up Defender for Cloud recommendations
- Investigate simulated attacks
- Create custom workbooks
- Configure automation workflows
