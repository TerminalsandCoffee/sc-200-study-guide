# Microsoft Entra ID (Azure AD)

Microsoft Entra ID Identity Protection, sign-in logs, audit logs, and identity security.

## Structure

- `identity-protection/` - Identity Protection risk detection and policies
- `sign-in-logs/` - Sign-in log analysis and investigation
- `audit-logs/` - Audit log analysis
- `conditional-access/` - Conditional Access policies and monitoring
- `privileged-identity/` - Privileged Identity Management (PIM)

## Key Topics

### Identity Protection
- User risk and sign-in risk
- Risk detections
- Risk policies (user risk, sign-in risk)
- Risk remediation
- Risk investigation

### Sign-in Logs
- Interactive sign-ins
- Non-interactive sign-ins
- Service principal sign-ins
- Managed identity sign-ins
- Sign-in diagnostics

### Audit Logs
- Directory changes
- Authentication changes
- Application changes
- Group changes
- Role assignment changes

### Conditional Access
- CA policy evaluation
- CA insights and workbook
- Report-only mode
- What-if tool
- Troubleshooting

### Privileged Identity Management
- Role assignments
- Access reviews
- Just-in-Time access
- Audit history

## Exam Focus

- Identity risk detection
