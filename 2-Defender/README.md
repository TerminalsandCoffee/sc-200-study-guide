# Microsoft Defender Suite

Microsoft Defender for Cloud, Defender for Endpoint, Defender for Identity, Microsoft 365 Defender, and Entra ID Identity Protection.

## Contents

- **`defender-cloud/`** – Defender for Cloud security posture, recommendations, and alerts
- **`defender-endpoint/`** – Defender for Endpoint (EDR) device management and threat detection
- **`m365-defender/`** – Microsoft 365 Defender (Email, Identity, Cloud Apps)
- **`entra-id/`** – Entra ID Identity Protection and monitoring

## Key Topics

- Cloud Security Posture Management (CSPM)
- Cloud Workload Protection Platform (CWPP)
- Endpoint detection and response
- Email and identity protection
- Cloud app security
- Identity risk detection
