# Microsoft Defender for Endpoint

Microsoft Defender for Endpoint (EDR) - endpoint detection, investigation, and response.

## Structure

- `device-management/` - Device onboarding and management
- `threat-hunting/` - Endpoint hunting queries and scenarios
- `incident-response/` - Endpoint incident investigation
- `vulnerability-management/` - Vulnerability assessment and remediation
- `configuration/` - Security policies and configuration

## Key Topics

### Device Management
- Device onboarding (Intune, GPO, script)
- Device groups and tags
- Security policies
- Device inventory
- Onboarding status

### Threat Detection
- Advanced threat protection
- Behavioral blocking
- Attack surface reduction
- Controlled folder access
- Network protection

### Investigation

- Network connections
- Alerts and incidents

### Threat Hunting
- Advanced hunting queries
- Custom detection rules
- Live response
- Automated investigation and remediation

### Vulnerability Management
