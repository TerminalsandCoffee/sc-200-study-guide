# Sentinel Workbooks

Custom workbooks and dashboards for Microsoft Sentinel visualization and analysis.

## Structure

- `security-overview/` - High-level security dashboards
- `threat-hunting/` - Hunting-focused workbooks
- `incident-investigation/` - Investigation and analysis workbooks
- `compliance/` - Compliance and audit workbooks
- `custom/` - Custom scenario-specific workbooks

## Workbook Types

### Security Overview
- Overall security posture
- Alert trends and statistics
- Top threats and entities
- Data connector health

### Threat Hunting
- Hunting query results
- Anomaly detection
- Behavioral analysis
- Threat intelligence overlays

### Incident Investigation
- Incident timeline
- Entity relationship graphs
- Query results and logs
- Investigation notes

### Compliance
- Audit log analysis
- Policy compliance
- Access review dashboards
- Security control effectiveness

## Exam Focus

- Creating custom workbooks
- Using KQL in workbooks
- Parameterization and interactivity
- Sharing and permissions
- Workbook templates
