# Start Here

Exam overview, roadmap, study plans, and general resources to get started with SC-200 preparation.

- **`roadmap/`** – Study plan, schedule, and exam objectives breakdown
- **`cheatsheets/`** – Quick reference guides and exam cram sheets (general)

1. Review the roadmap and exam objectives
2. Familiarize yourself with the exam domains
3. Use cheatsheets as quick reference during study
