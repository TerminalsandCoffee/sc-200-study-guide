# Study Notes

Domain-based notes organized by SC-200 exam areas. Capture key concepts, configurations, and learnings from docs, labs, videos, and hands-on practice.

## Structure

Organize notes by exam domain:
- `sentinel/` - Microsoft Sentinel architecture, KQL, analytics rules
- `defender-cloud/` - Defender for Cloud security posture, recommendations
- `m365-defender/` - Email, Identity, Cloud Apps protection
- `defender-endpoint/` - EDR, threat hunting, device management
- `entra-id/` - Identity Protection, sign-in logs, risk detection
- `incident-response/` - Investigation workflows, response procedures
- `automation/` - Logic Apps, playbooks, SOAR concepts

## Note-Taking Tips

- Use clear headings and structure
- Include code snippets and KQL examples
- Document common pitfalls and gotchas
- Link to official documentation
- Add diagrams/screenshots where helpful
- Review and update notes regularly
