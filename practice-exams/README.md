# Practice Exams

Practice questions, exam simulations, and assessment materials for SC-200 preparation.

## Structure

- `questions/` - Practice questions organized by domain
- `simulations/` - Full exam simulations
- `flashcards/` - Study flashcards and quick review
- `assessments/` - Self-assessments and progress tracking

## Exam Format

- **Duration**: 100-120 minutes
- **Question Types**: Multiple choice, case studies, drag-and-drop
- **Passing Score**: ~700/1000 (varies)
- **Domains**: 4 main domains with varying weights

## Study Approach

1. **Domain Review** - Study each domain thoroughly
2. **Practice Questions** - Answer questions by domain
3. **Full Simulations** - Take timed practice exams
4. **Weak Areas** - Focus on domains with lower scores
5. **Final Review** - Review all domains before exam
