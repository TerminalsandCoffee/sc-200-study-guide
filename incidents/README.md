# Incident Response

Incident response procedures, templates, workflows, and case studies for Microsoft Sentinel and Microsoft 365 Defender.

## Structure

- `procedures/` - Incident response procedures and runbooks
- `templates/` - Incident templates and checklists
- `case-studies/` - Real-world incident scenarios and walkthroughs
- `investigation/` - Investigation techniques and methodologies
- `playbooks/` - Response playbooks and workflows

## Incident Response Lifecycle

1. **Preparation** - Tools, procedures, and team readiness
2. **Detection & Analysis** - Identifying and analyzing security events
3. **Containment** - Short-term and long-term containment strategies
4. **Eradication** - Removing threats from the environment
5. **Recovery** - Restoring systems and services
6. **Post-Incident Activity** - Lessons learned and improvements

## Key Topics

### Investigation
- Entity investigation

### Response Actions
- Account actions (disable, reset password)
