# SC-200 Study Guide

SC-200 study guide + hands-on detection engineering labs.

This repository documents my path to passing the Microsoft SC-200 (Security Operations Analyst) exam and building detection engineering skills.

## Goals

- Pass SC-200 by **end of January 2025**
- Build real Sentinel + KQL detection labs
- Detection Engineering foundation
- Hands-on experience with Microsoft security stack

## Repository Structure

- **`0-Start-Here/`** – Exam overview, roadmap, study plans, cheatsheets, and practice exams
- **`1-Sentinel/`** – Microsoft Sentinel architecture, data connectors, analytics rules, workbooks, playbooks, incidents, and automation
- **`2-Defender/`** – Defender for Cloud, Defender for Endpoint, Defender for Identity, Microsoft 365 Defender, and Entra ID
- **`3-KQL/`** – KQL fundamentals, hunting queries, investigation patterns, and query libraries
- **`4-Labs/`** – Hands-on labs, incident simulations, automation workflows, scripts, and scenarios
