# Microsoft 365 Defender

Microsoft 365 Defender suite: Email, Identity, Cloud Apps, and Office 365 protection.

## Structure

- `defender-for-email/` - Microsoft Defender for Office 365
- `defender-for-identity/` - Microsoft Defender for Identity
- `defender-for-cloud-apps/` - Microsoft Defender for Cloud Apps
- `defender-for-office365/` - Office 365 protection features
- `unified-experience/` - Microsoft 365 Defender portal

## Key Topics

### Defender for Office 365
- Safe Links and Safe Attachments
- Anti-phishing policies
- Anti-spam and anti-malware
- Threat Explorer
- Attack simulation training

### Defender for Identity
- Active Directory monitoring
- Lateral movement detection
- Pass-the-hash detection
- Suspicious activities
- Identity security posture

### Defender for Cloud Apps
- Cloud Discovery
- App governance
- Conditional Access App Control
- Shadow IT detection
- Data loss prevention (DLP)

### Unified Experience
- Microsoft 365 Defender portal
- Incident management
- Advanced hunting
- Threat analytics
- Action center
